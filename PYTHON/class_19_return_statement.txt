-----------------
Function can take input values as parameter and excecutes business logic and returns output to the caller with 'return' statements.

    return x+y

result = add(30,70)

Q_2:- Write a function to check whether the given number is even or odd?

def even_odd(num):
    if num % 2 == 0:
        print(num,'is even number')
    else:
        print(num,'is odd number')

even_odd(15)
